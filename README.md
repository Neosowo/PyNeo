# NeoPY - Plataforma de Aprendizaje Interactivo

## Descripción

NeoPY es una plataforma de aprendizaje interactivo diseñada para enseñar programación en Python desde cero. Utiliza visualizaciones gráficas, ejercicios prácticos y una interfaz moderna para facilitar el aprendizaje.

## Características

- **8 Módulos Progresivos**: Desde conceptos básicos hasta proyectos completos
- **Visualizaciones Interactivas**: Cada concepto se explica con ejemplos visuales
- **Ejercicios Prácticos**: Practica lo aprendido con ejercicios interactivos
- **Sistema de Progreso**: Guarda automáticamente tu avance
- **Diseño Moderno**: Interfaz premium con glassmorphism y animaciones suaves
- **100% Gratis**: Sin costos ni registros

## Módulos

1. **Qué es Programar** - Introducción a la programación
2. **Tipos de Datos** - Números, textos y booleanos
3. **Operaciones Básicas** - Matemáticas en Python
4. **Condicionales** - Tomar decisiones con if/else
5. **Bucles** - Repetir tareas con for
6. **Listas** - Colecciones de datos
7. **Funciones** - Reutilizar código
8. **Proyecto Final** - Aplicar todo lo aprendido

## Tecnologías

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter, JetBrains Mono)

## Instalación

1. Clona o descarga este repositorio
2. Abre `index.html` en tu navegador
3. Comienza a aprender

No requiere instalación de dependencias ni servidor.

## Uso

1. Navega por los módulos en orden secuencial
2. Completa cada lección para desbloquear la siguiente
3. Practica con los ejercicios interactivos
4. Tu progreso se guarda automáticamente en localStorage

## Características Técnicas

- **Glassmorphism**: Efectos de vidrio esmerilado
- **Animaciones 3D**: Tarjetas con efecto tilt
- **Partículas Animadas**: Fondo dinámico
- **Responsive Design**: Adaptable a todos los dispositivos
- **Dark Theme**: Diseño oscuro profesional
- **Progress Tracking**: Sistema de seguimiento de progreso
